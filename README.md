
🏦 MyBank - Simple Banking System (Streamlit App)
A simple and beginner-friendly Bank Management System built using Python + Streamlit.
You can create accounts, deposit money, withdraw money, update details, and even delete accounts.

🚀 Features
🔹 Create Account
Auto-generated Account Number
Stores Name, Email, Phone, PIN
Saves everything in database.json
🔹 Deposit Money
Deposit up to ₹10,000
PIN + Account No. verification
🔹 Withdraw Money
Withdraw up to ₹10,000
Shows remaining balance
🔹 View Account Details
Full account info with balance display
🔹 Update Details
Change Name / Email / Phone / PIN
🔹 Delete Account
Remove account permanently
📦 Project Structure
Bank management/ │ ├── app.py # Main Streamlit app ├── database.json # Stores account data ├── requirements.txt # Python libraries required └── README.md # Project documentation (this file)

🛠️ How to Run Locally
Install Streamlit: pip install streamlit

Run the app: streamlit run app.py

🌐 Deployment
This project is deployed using Streamlit Community Cloud.
Simply connect the GitHub repo → Click Deploy.
